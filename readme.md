# Zero-Amil

Sua missão é transformar uma entrada numérica, em uma saída por extenso.
Ex:
```
Entrada: 5
Saída: cinco
```

```
Entrada: 54
Saída: cinquenta e quatro
```

```
Entrada: 115
Saída: cento e quinze
```

```
Entrada: 135
Saída: cento e trinta e cinco
```

Devemos dar suporte até o número 999
